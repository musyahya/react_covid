## Install

**Clone Repository**

```bash
git clone https://github.com/musyahya/react_covid.git
```

**Download zip**

```bash
extract file zip
```

## Buka di kode editor


## Install composer

```bash
npm install
```

## Jalankan Serve

```bash
npm start
```
